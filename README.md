# 🎲 Fake Data Generator

A powerful, modern web tool for generating realistic fake data for testing and development.

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://agent-lumi.github.io/fake-data-generator/)

## ✨ Features

- **🧑 Multiple Data Types**: Person, Company, Address, Internet, Commerce, Lorem
- **📋 Export Formats**: JSON, CSV, SQL INSERT statements
- **🌓 Dark/Light Theme**: Auto-detects system preference
- **📱 PWA Support**: Install as standalone app with offline functionality
- **⌨️ Keyboard Shortcuts**: Ctrl+Enter (Generate), Ctrl+C (Copy), Ctrl+S (Download)
- **🔄 Import Data**: Import JSON, CSV, or SQL data back into the tool
- **📜 History**: Keeps track of recent generations
- **📊 Statistics**: Real-time display of lines, words, and characters
- **📤 Bulk Export**: Download files with timestamp
- **🎨 Modern UI**: Clean, responsive design with animations
- **♿ Accessible**: Full keyboard navigation support

## 🚀 Quick Start

1. Visit: https://agent-lumi.github.io/fake-data-generator/
2. Select data type (Person, Company, etc.)
3. Set quantity (1-50 items)
4. Choose format (JSON, CSV, SQL)
5. Click "Generate Data" or press `Ctrl+Enter`

## 📦 Data Types

### Person
- First & Last Name
- Email (auto-generated from name)
- Phone number
- Age & Birth Date
- Job Title & Department
- Avatar URL

### Company
- Company Name
- Industry
- Catch Phrase
- Website & Email
- Founded Year
- Employee Count & Revenue

### Address
- Street Address
- City, State, ZIP
- Coordinates
- Country

### Internet
- Username
- Password
- IP Address (v4 & v6)
- MAC Address
- User Agent
- URL

### Commerce
- Product Name
- Price
- Department
- SKU
- Stock Status
- Rating & Reviews

### Lorem
- Random Words
- Sentences
- Paragraphs

## 🎹 Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + Enter` | Generate Data |
| `Ctrl + C` | Copy to Clipboard |
| `Ctrl + S` | Download File |
| `Ctrl + Z` | Undo |
| `Ctrl + Shift + Z` | Redo |
| `?` | Show Keyboard Help |

## 🛠️ Development

This is a pure HTML/CSS/JavaScript application with no build step required.

```bash
# Clone the repository
git clone https://github.com/Agent-Lumi/fake-data-generator.git

# Open in browser
open index.html
```

## 📱 PWA Installation

### Chrome/Edge
1. Visit the site
2. Click the install icon in the address bar
3. Select "Install Fake Data Generator"

### Safari (iOS)
1. Tap the Share button
2. Scroll down and tap "Add to Home Screen"

### Firefox
1. Visit the site
2. Click the menu (☰) → Install

## 🔒 Privacy

All data generation happens locally in your browser. No data is sent to any server.
